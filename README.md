# BOAT AI LAB v5

## 概要

BOAT AI LAB v5 は、ボートレースの出走表・展示・モーター・気象・オッズなどの情報を総合的に分析し、根拠に基づいた考察を生成するAI分析システムです。

## 目的

- データに基づくレース分析
- 各選手・モーター・展示気配の評価
- スコアリングによる客観的比較
- レース展開の考察
- 本命から穴まで根拠を提示

## プロンプト構成

- ANALYSIS_RULES.md
- OUTPUT_FORMAT.md
- SCORING_SYSTEM.md
- MOTOR_ENGINE.md
- START_ANALYSIS.md
- DATA_INPUT.md
- BOAT_AI_LAB_v5.md

## 基本フロー

1. データを受け取る
2. 不足データを確認
3. モーター評価
4. 展示評価
5. スコアリング
6. 展開予測
7. 買い目候補
8. 最終レース評価
